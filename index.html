java-microedition-libraries

Libraries for the Java Microedition

A collection of libraries for the development of Java Microedition applications:

  • Stubs of APIs (CLDC, IMP, MIDP, CDC) to compile against
  • Lib-Cross-Java: Java Microedition extension library
  • SLF for CLDC: Simple Logging Facade for CLDC 1.1
